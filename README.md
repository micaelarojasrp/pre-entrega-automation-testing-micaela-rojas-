
# pre-entrega-automation-testing-micaela-rojas


**Propósito:** Proyecto de automatización para la pre-entrega del curso. Automatiza flujos básicos en `www.saucedemo.com` usando Selenium + Pytest.


**Tecnologías:**
- Python 3.8+
- Selenium
- Webdriver-Manager
- Pytest
- pytest-html


**Estructura:**
- `tests/` : casos de prueba (pytest)
- `utils/` : funciones auxiliares reutilizables
- `conftest.py` : fixtures y hooks pytest
- `reports/` : reportes HTML y capturas
